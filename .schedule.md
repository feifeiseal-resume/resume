## 主要作品目標

### 主作品

JOSUI - shadcn/ui 色彩主題預覽工具

- Website: https://josui.space/
- GitHub: https://github.com/FeiFeiSeal/josui

### 作品定位

JOSUI 是一個針對 shadcn/ui 使用者設計的色彩主題預覽工具，協助開發者與設計師在導入設計系統時，更快確認色彩 token 在實際元件上的呈現效果。

這個作品用來證明：

- 具備 React / TypeScript / Tailwind CSS / shadcn-ui 生態經驗
- 能從實務問題出發設計工具
- 理解設計系統、色彩 token、元件展示與前端體驗
- 能獨立部署與維護公開產品
- 雖然主要求職方向是 Vue / Nuxt，但具備跨框架學習與實作能力

### 近期不做事項

- 不新增大型功能
- 不重構架構
- 不建立新的 Job Tracker 專案
- 不投入 React / Next.js 大轉換
- 不公開公司專案程式碼
- 不建立有 NDA 風險的公開案例頁
