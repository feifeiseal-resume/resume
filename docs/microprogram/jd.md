▍關於這個職位

我們正在尋找一位對產品開發充滿熱情的 Junior 前端工程師。你將參與產品從需求討論、開發、測試到上線的完整流程，並透過 AI 開發工具提升效率，在實務專案中累積現代軟體工程能力。

我們重視團隊合作、知識分享與持續改善，期待你樂於學習、願意接受回饋，並對打造優質產品有高度熱情。歡迎對前端開發、自動化流程與 AI 工具應用有興趣，並樂於在實務中持續學習與成長的夥伴加入我們！

▍工作內容

1.  前端產品開發與維護：完成公司交付之 Web 前端功能開發、調整與維護，並持續優化既有專案的功能與使用者體驗。

2. 技術與風險評估：協助進行技術可行性、風險與工時評估，並參與技術方案討論。

3. 需求分析：協助拆解產品需求，轉換為技術方案與 AI 實作程式碼的文件，專注於系統設計、測試驗收與可觀測性建置，落實高品質交付。

4. AI 產出審查：參與 AI 產出程式碼的檢視與驗證，辨識潛在問題，並確保程式品質符合團隊標準。

5. 開發品質優化：參與 Code Review，維持程式碼品質與可維護性；協助 CI 流程維護與優化，落實前端資訊安全控管與文件撰寫等開發實務。

▍必備條件

1. 1 年以上前端開發經驗，熟悉 HTML / CSS / JavaScript 開發。
2. 具備 React 或 Vue、TypeScript，能獨立開發與維護前端專案。
3. 有撰寫單元測試的習慣，重視程式碼的可測試性與可維護性。
4. 使用過 AI 輔助開發工具（如 Claude Code、Cursor、Codex）。
5. 熟悉 Git 版本控制，並具備 Docker 容器化的基本概念與操作經驗。

▍加分條件

1. 有撰寫單元測試的習慣，重視程式碼的可測試性與可維護性。
2. 曾使用 MCP、AI Agent、Agent Skills 相關應用經驗。
3. 具備 React Native App 開發經驗。
4. 具備 Electron App 開發經驗。

▍人格特質

1. 注重細節：對使用者體驗有敏銳洞察力，在意載入速度、互動順暢度與程式碼的可維護性。
2. 擅於溝通：樂於與產品、設計、後端、測試等不同角色協作，並在 code review 中坦誠交流。
3. 主動積極：能評估技術方案的可行性與成本，勇於排查問題並提出改善做法。
4. 擁抱改變：對 AI 工具與新技術保持好奇，樂於學習並快速應用在實際開發中。
